Iara (Sim, um trocadilho com "IA") é uma assistente de recepção virtual para coletar informações de visitantes e agendar consultas médicas de forma eficiente.

## Funcionalidades

- Coleta de informações do visitante, como nome, contato e motivo da visita.
- Agendamento automático de consultas médicas. (TBD)
- Integração com calendários para evitar conflitos de horários. (TBD)
- Envio de lembretes de consultas via e-mail ou SMS. (TBD)

## Nota

É UM PROJETO EXPERIMENTAL! O uso de tokens é alto intencionalmente para testes e desenvolvimento. Não é recomendado para uso em produção.

## Tecnologias Utilizadas

- Python 3.13
- Pydantic
- Google GenAI SDK
